# Security policy

## Supported versions

安全更新仅针对最新 GitHub Release 和 `main` 分支。

## Reporting a vulnerability

请通过 [GitHub 私密漏洞报告](https://github.com/dcjaychou-design/NaiWaPet/security/advisories/new) 提交漏洞，不要先公开能够被直接利用的细节。报告中请包含受影响版本、Windows 版本、复现步骤和可能影响。

本项目不会联网、不会收集遥测；设置仅写入 `%LOCALAPPDATA%\NaiWaPet\settings.json`，开机启动仅使用当前用户的 Windows `Run` 注册表项。
