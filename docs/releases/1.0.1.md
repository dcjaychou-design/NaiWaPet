# NaiWaPet 1.0.1

NaiWaPet 是一款适用于 Windows 11 x64 的透明奶蛙桌宠。本版本继续提供可直接运行的单文件程序和便携压缩包，无需另外安装 .NET 运行时；动画、声音、拖动、抛掷、鼠标穿透和开机启动方式保持不变。

## 下载

- `NaiWaPet-1.0.1-win-x64.exe`：推荐，下载后直接双击运行。
- `NaiWaPet-1.0.1-win-x64-portable.zip`：解压后运行其中的 `NaiWaPet.exe`。
- 两个同名 `.sha256` 文件：分别用于核对 EXE 和 ZIP 是否完整。

## 本次改进

- 新增“关于与开源许可”，可从托盘菜单或设置窗口查看程序版本、项目许可及完整的 .NET/WPF 运行时许可。
- 便携 ZIP 增加 .NET 10.0.10 与 WPF 10.0.10 的官方许可和第三方声明。
- 固定 .NET 10 SDK 10.0.302 与自包含运行时 10.0.10，并加强代码分析和依赖安全检查。
- 发布前完整检查 EXE、ZIP、许可文件、Windows 文件版本和 SHA-256，且确认 ZIP 内 EXE 与独立 EXE 完全一致。
- 发布文件先以草稿上传并重新下载复核，通过后才公开；公开后标签和文件不可替换，并带有 GitHub Release 证明。

## 使用与隐私

- 支持 Windows 11 x64。
- 程序默认播放原视频笑声，可在托盘菜单或设置中关闭。
- 原有 Schema 2 设置继续使用，不会因升级而迁移或清除。
- 设置保存在 `%LOCALAPPDATA%\NaiWaPet\settings.json`，程序不连接网络、不收集遥测，也不上传使用数据。
- 本项目未购买商业代码签名证书，Windows SmartScreen 首次运行时仍可能显示“未知发布者”。不可变发布和发布证明可以核验文件来源，但不会消除该提示。

## 校验下载文件

在 PowerShell 中运行：

```powershell
Get-FileHash .\NaiWaPet-1.0.1-win-x64.exe -Algorithm SHA256
Get-Content .\NaiWaPet-1.0.1-win-x64.exe.sha256
```

两处显示的 SHA-256 值应当相同。便携 ZIP 可以用相同方法核对。安装 GitHub CLI 后，还可以运行 `gh release verify v1.0.1 --repo dcjaychou-design/NaiWaPet` 验证发布证明。
