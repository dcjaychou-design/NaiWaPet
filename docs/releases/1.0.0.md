# NaiWaPet 1.0.0

NaiWaPet 是一款适用于 Windows 11 x64 的透明奶蛙桌宠。本发布提供可直接运行的单文件程序和便携压缩包，无需另外安装 .NET 运行时。

本次发布物已于 2026 年 7 月 22 日从 `v1.0.0` 标签对应的完整源码重新构建，在保留 1.0.0 版本号和原有功能的同时，纳入了首发后的稳定性与完整性改进。

## 下载

- `NaiWaPet-1.0.0-win-x64.exe`：推荐，下载后直接双击运行。
- `NaiWaPet-1.0.0-win-x64-portable.zip`：解压后运行其中的 `NaiWaPet.exe`。
- 两个同名 `.sha256` 文件：分别用于核对 EXE 和 ZIP 是否完整。

## 本次完善

- 降低动画切换和素材检查时的内存峰值。
- 加强动画清单、点击区域、源文件路径和素材哈希校验。
- 改进单实例启动、异常日志、设置保存和开机启动状态处理。

## 使用提示

- 支持 Windows 11 x64。
- 程序默认播放原视频笑声，可在托盘菜单或设置中关闭。
- 设置保存在 `%LOCALAPPDATA%\NaiWaPet\settings.json`，程序不会收集遥测或上传使用数据。
- 本项目暂未购买商业代码签名证书，Windows SmartScreen 首次运行时可能显示“未知发布者”。请确认文件来自本发布页面，并使用 `.sha256` 文件核对哈希。

## 校验下载文件

在 PowerShell 中运行：

```powershell
Get-FileHash .\NaiWaPet-1.0.0-win-x64.exe -Algorithm SHA256
Get-Content .\NaiWaPet-1.0.0-win-x64.exe.sha256
```

两处显示的 SHA-256 值应当相同。便携 ZIP 可以用相同方法核对。
