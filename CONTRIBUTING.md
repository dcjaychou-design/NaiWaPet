# 参与贡献

感谢你帮助改进奶蛙桌宠。

## 开发流程

1. Fork 仓库并从 `main` 创建功能分支。
2. 使用 Windows 11 x64 和 .NET 10 SDK 开发。
3. 运行 `./build.ps1`，确保编译、核心测试和素材校验全部通过。
4. 如果修改了动画生成逻辑，按照 `docs/BUILDING.md` 重新生成素材，并检查透明边缘和所有阶段的动作。
5. 提交说明应解释用户可见变化；Pull Request 不要混入无关格式化。

## 素材贡献

提交新图片、视频或声音时，必须同时说明可核验来源、许可、作者或生成方式，并提供原始文件 SHA-256。不要提交来源不明或声称“网上公开所以随便用”的素材。

## 代码风格

- C# 开启 Nullable、隐式 using 和 warnings-as-errors。
- UI 文字优先使用简体中文，代码标识符使用英文。
- 核心算法尽量放入 `NaiWaPet.Core` 并补充测试。
- 保持普通用户无需管理员权限、无需预装运行库。
